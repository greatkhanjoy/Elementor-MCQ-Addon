Create engaging quizzes with the MCQ, Quiz, Question Addon for Elementor.

== Description ==

MCQ, Quiz, Question Addon is a lightweight extension for Elementor, enabling you to effortlessly integrate quizzes and questions into your WordPress site. Enhance user engagement and interaction with customizable quiz options.

== Features ==

- Easily integrate quizzes into your Elementor pages.
- Create multiple-choice questions effortlessly.
- Customize quiz styles and appearance.
- Requires Elementor plugin for optimal functionality.

== Installation ==

1. Upload the `mcq-quiz-question-addon` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure Elementor is installed and activated.

== Usage ==

1. After installation, you will find the MCQ, Quiz, Question widget in the Elementor editor.
2. Drag and drop the widget onto your desired section.
3. Customize quiz settings, questions, and styles within the Elementor editor.
